# 05 Field Mapping

## Purpose

This document defines sample field ownership and output mapping.

## Sample Ownership Logic

OCR owns initial extraction.
Owner owns business confirmation for Non-PO invoices.
PO / GR data owns matching context for PO invoices.
AP owns final review and posting readiness.
Workflow owns status, audit trail, and export tracking.

## Sample Output Fields

- SAMPLE_COMPANY_CODE
- SAMPLE_VENDOR_ID
- SAMPLE_INVOICE_NUMBER
- SAMPLE_INVOICE_DATE
- SAMPLE_COST_CENTER
- SAMPLE_INTERNAL_ORDER
- SAMPLE_TAX_CODE
- SAMPLE_AMOUNT
