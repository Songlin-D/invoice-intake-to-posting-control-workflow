# 03 Non-PO Workflow

## Purpose

This document describes the Non-PO invoice path.

## Process

Invoice PDF
-> OCR Extraction
-> Draft Request
-> Owner Confirmation
-> Approval Workflow
-> AP Review
-> SAMPLE_NON_PO_UPLOAD_TEMPLATE

## Control Logic

Non-PO invoices require business confirmation, approval evidence, AP review, and SAP output preparation using a Non-PO upload structure.
