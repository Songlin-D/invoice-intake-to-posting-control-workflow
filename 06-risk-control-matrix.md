# 06 Risk and Control Matrix

## Purpose

This document summarizes key workflow risks and control responses.

## Key Risks

- OCR extraction error
- Wrong invoice classification
- Missing approval evidence
- PO invoice routed as Non-PO
- No GR or service entry for PO invoice
- Offline exception handling
- Weak audit trail

## Control Responses

- OCR confidence check
- AP triage for unclear invoices
- Approval history retention
- Separate PO and Non-PO routing
- Exception queue and owner assignment
- AP final review before SAP output
